# สรุปร่างเอกสารค่าตอบแทนกรรมการคุมสอบและกรรมการจ่ายข้อสอบ

**English title**: EMS Payment Document Draft Review One-Pager  
**วันที่จัดทำ**: 2026-06-08  
**สถานะเอกสาร**: `DRAFT_NOT_AUTHORIZED`  
**สถานะการตรวจทาน**: `PENDING_SUPERVISOR_FINANCE_REVIEW`

## 1. วัตถุประสงค์

EMS ได้จัดเตรียมหน้าร่างเอกสารสรุปค่าตอบแทนรูปแบบใกล้เคียงเอกสารทางการของภาคเรียน `2/2568` เพื่อให้ผู้บังคับบัญชา/ฝ่ายการเงินตรวจทานก่อนตัดสินใจขั้นถัดไป

เอกสารนี้มีไว้เพื่อช่วยตอบว่า:

- รูปแบบตารางร่างตรงกับตัวอย่างทางการหรือไม่
- ควรใช้อัตราค่าตอบแทนชุดใด
- ควรใช้แหล่งข้อมูลใดสำหรับจำนวนกรรมการจ่ายข้อสอบ
- สามารถออกแบบ workflow ร่าง export ในอนาคตได้หรือไม่

## 2. สถานะปัจจุบัน

- มีหน้าร่างในระบบที่ `/invigilation-payment-document-draft`
- UI ผ่านการจัดรูปแบบและอยู่ในสถานะ `HUMAN_VISUAL_QA_PASSED_ACCEPTED_FOR_SUPERVISOR_REVIEW`
- สถานะเอกสารยังเป็น `DRAFT_NOT_AUTHORIZED`
- ยังไม่มีปุ่มหรือกระบวนการอนุมัติจ่ายเงิน
- ยังไม่มีการอนุมัติขั้นสุดท้าย
- ยังไม่มี official PDF / Excel / export

## 3. หลักฐานอ้างอิงปัจจุบัน

- ใช้ตัวอย่างเอกสารสรุปรูปแบบทางการของภาคเรียน `2/2568` เป็นแนวทางร่าง
- อัตราจากตัวอย่าง 2/2568:
  - วันจันทร์-ศุกร์: `120 THB` ต่อคน/ครั้ง
  - วันเสาร์-อาทิตย์: `200 THB` ต่อคน/ครั้ง
- อัตรา active `300/500` ในระบบยังถือเป็น demo/test data เท่านั้นสำหรับเส้นทางร่างนี้

หมวดหน้าที่ที่ต้องแสดงในเอกสารร่าง:

- `INVIGILATION_COMMITTEE` / กรรมการคุมสอบ
- `PAPER_DISTRIBUTION_COMMITTEE` / กรรมการจ่ายข้อสอบ

## 4. การตัดสินใจที่ต้องการ

- ยอมรับรูปแบบเอกสารร่างหรือไม่
- ยืนยันอัตราค่าตอบแทนสำหรับภาคเรียน `2/2568`
- ยืนยันแหล่งข้อมูลจำนวนกรรมการจ่ายข้อสอบ
- อนุญาตให้ทีมออกแบบ workflow ร่าง export ในอนาคตหรือไม่
- ระบุสิ่งที่ต้องแก้ไขหรือพักไว้ก่อนดำเนินการต่อ

## 5. สิ่งที่การตรวจทานนี้ไม่ใช่

- ไม่ใช่การอนุมัติจ่ายเงิน
- ไม่ใช่เอกสารเบิกจ่ายขั้นสุดท้าย
- ไม่ใช่ official PDF / Excel / export
- ไม่ใช่การยืนยันข้อมูลเป็น final truth
- ไม่ใช่การเปลี่ยนอัตรา active ใน EMS
- ไม่ใช่การเพิ่ม production readiness

ข้อสรุปจากการตรวจทานต้องถูกบันทึกในแบบฟอร์มตัดสินใจของผู้บังคับบัญชา/ฝ่ายการเงินก่อนเริ่มงานขั้นถัดไป
