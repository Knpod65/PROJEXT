# ตัวอย่างอ้างอิงสรุปค่าตอบแทน 2/2568

**English title**: Official-Style 2/2568 Payment Summary Sample Reference
**สถานะหลักฐาน**: `USER_PROVIDED_TRANSCRIPTION_PENDING_PROVENANCE_VERIFICATION`
**สถานะการตัดสินใจ**: `PENDING_FINANCE_ADMIN_REVIEW`

## ข้อมูลอ้างอิงที่ถอดความจากผู้ใช้

- ชื่อเอกสาร: `สรุปจำนวนกรรมการและค่าตอบแทน รายวัน/ช่วงเวลา`
- ภาคการศึกษา: `ภาคการศึกษาที่ 2/2568`
- อัตราวันจันทร์-ศุกร์: `120 บาท/คน/ครั้ง`
- อัตราวันเสาร์-อาทิตย์: `200 บาท/คน/ครั้ง`

## การตรวจสอบความสอดคล้องทางคณิตศาสตร์

- กรรมการคุมสอบ: จำนวนวันธรรมดา 109 และวันเสาร์-อาทิตย์ 36 จะได้ `109 × 120 + 36 × 200 = 20,280 THB`
- กรรมการจ่ายข้อสอบ: จำนวนวันธรรมดา 23 และวันเสาร์-อาทิตย์ 6 จะได้ `23 × 120 + 6 × 200 = 3,960 THB`
- ยอดรวม: `20,280 + 3,960 = 24,240 THB`

จำนวนแยกวันข้างต้นเป็นค่าที่อนุมานจากยอดรวม ไม่ใช่ข้อมูลรายแถวที่ตรวจสอบจากต้นฉบับ

## ข้อจำกัดของหลักฐาน

- ไม่พบไฟล์ภาพต้นฉบับใน attachment directory ของ pass นี้
- ยังไม่ตรวจสอบรายการรายวัน/ช่วงเวลารายแถว
- ยังไม่ยืนยันผู้จัดทำ เลขที่เอกสาร วันที่อนุมัติ หรือผู้ลงนาม
- ยังไม่ยืนยันว่าอัตรา `120/200` ใช้กับการดำเนินงานปัจจุบัน
