# เช็กลิสต์สั้นสำหรับผู้บังคับบัญชา/ฝ่ายการเงิน

**English title**: Supervisor / Finance Quick Review Checklist  
**สถานะเอกสาร**: `DRAFT_NOT_AUTHORIZED`  
**หมายเหตุ**: เช็กลิสต์นี้ไม่ใช่การอนุมัติจ่ายเงิน

## ตรวจรูปแบบร่างเอกสาร

- [ ] รูปแบบตารางใกล้เคียง summary style ของตัวอย่าง 2/2568
- [ ] มีทั้งหมวด `INVIGILATION_COMMITTEE` / กรรมการคุมสอบ
- [ ] มีทั้งหมวด `PAPER_DISTRIBUTION_COMMITTEE` / กรรมการจ่ายข้อสอบ
- [ ] จัดกลุ่มตามวันที่สอบและช่วงเวลาอย่างเข้าใจง่าย
- [ ] แสดงประเภทวันชัดเจน

## ตรวจอัตราและยอดรวม

- [ ] อัตราที่ใช้ในร่างตรงกับ decision ที่ผู้ตรวจทานต้องการ
- [ ] ถ้าใช้ตัวอย่าง 2/2568: วันจันทร์-ศุกร์ `120 THB`
- [ ] ถ้าใช้ตัวอย่าง 2/2568: วันเสาร์-อาทิตย์ `200 THB`
- [ ] ยอดรวมแยกหมวดและยอดรวมทั้งเอกสารอ่านง่าย

## ตรวจแหล่งข้อมูล

- [ ] จำนวนกรรมการคุมสอบมาจากข้อมูลที่เข้าเกณฑ์ร่าง
- [ ] จำนวนกรรมการจ่ายข้อสอบมีแหล่งข้อมูลที่ผู้ตรวจทานยอมรับได้
- [ ] ถ้าแหล่งข้อมูลยังไม่ชัดเจน ให้เลือก hold/validate source ก่อน

## ตรวจสถานะความปลอดภัย

- [ ] เห็นหรือรับทราบสถานะ `DRAFT_NOT_AUTHORIZED`
- [ ] ไม่มี final approval button
- [ ] ไม่มี official export button
- [ ] ไม่มี PDF/Excel export button
- [ ] ไม่มีข้อความที่สื่อว่าเป็น payment authorization

## สรุปขั้นถัดไป

- [ ] พร้อมให้ทีมออกแบบ draft export workflow ในอนาคต
- [ ] ต้องแก้รูปแบบก่อน
- [ ] ต้อง validate paper-distribution source ก่อน
- [ ] ต้องรอ rate decision ก่อน
- [ ] ต้องหยุดและ redesign
