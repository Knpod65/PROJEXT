# Role Manuals

## Purpose

Each role manual explains how a person should use EMS in daily practice.

The manual format is intentionally consistent so users can move between roles without relearning the structure of the guide.

## Standard Manual Structure

Each role manual should answer these questions:

1. What is this role for?
2. What does a normal day look like?
3. What dashboards or pages matter most?
4. What decisions can this role make?
5. What warnings should this role watch for?
6. What mistakes are common?
7. When should this role escalate?
8. What governance responsibility does this role carry?
