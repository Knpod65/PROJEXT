# Visual Journeys

## Purpose

Each journey is a step-by-step path through a real EMS workflow.

The journey format should show the screen sequence, the decision points, the operational meaning, and the escalation path.

## Journey Structure

Each journey should answer:

1. Where do I start?
2. What screen comes next?
3. What am I supposed to notice?
4. What action do I take?
5. What happens if something looks wrong?

## Capture Rule

The screenshot sequence should follow the workflow in order.

Do not capture isolated screens without explaining the path between them.
