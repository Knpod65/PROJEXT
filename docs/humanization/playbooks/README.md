# Operational Playbooks

## Purpose

Playbooks describe how EMS should behave during known operating conditions.

They help people act quickly during routine, peak, and failure scenarios.

## Starter Playbooks

- Normal operations
- Peak exam periods
- Operational overload
- Room conflicts
- Distribution failures
- Governance escalation
- System degradation
- Workload imbalance
- Staffing shortages
- Emergency continuity

## Playbook Rule

Every playbook should define:

1. What the situation looks like
2. What should happen first
3. Who owns the response
4. When to escalate
5. What success looks like
