# National Intelligence Guide

## Purpose

National Intelligence gives a high-level view of cross-institution pressure, resilience, and coordination risk.

It is useful when the system must be understood at a broader scale than one institution.

## What Matters Most

- Cross-unit pressure patterns
- Shared risk signals
- Resilience trends
- Coordination gaps
- Whether the issue is local or systemic

## Metric Interpretation

- A repeated pattern across units may indicate a structural issue
- A local spike may only require one institution to respond
- A resilience signal shows how well the broader system can absorb pressure
- A coordination signal shows whether multiple units need to align

## Urgency Levels

- Green: monitor normally
- Amber: review the shared pattern
- Red: coordinate a response across units

## Warning Interpretation

- Shared warnings mean the issue is larger than one local workflow
- A national view should not erase institutional boundaries
- Forecast warnings should be treated as planning signals

## Operational Meaning

This dashboard answers: is there a wider system pattern we need to pay attention to?

## Governance Meaning

National-level visibility exists to support continuity, resilience, and shared decision-making.

## Recommended Actions

1. Check whether the signal is local or cross-system.
2. Review which institutions are involved.
3. Confirm whether the pattern is repeated.
4. Escalate if the issue affects continuity, coordination, or resilience.

## Escalation Triggers

- Multi-institution resilience decline
- Shared staffing or readiness pressure
- Repeated cross-unit failure pattern
- Coordination gap that cannot be solved locally

## Common Misunderstandings

- National visibility is not the same as national control.
- A local issue does not automatically become systemic.
- A forecasted pattern still needs context before action.

## Simplified Explanation

This dashboard helps coordinators see whether the broader education system is stable or under strain.
