# Claude Design Handoff Package

## 1. Purpose

This is the single folder to send to Claude Design, Figma, or any design agent for EMS redesign work.

## 2. How To Use

Recommended order:

1. Read `00_CLAUDE_DESIGN_MASTER_PROMPT.md`
2. Read `01_EMS_DESIGN_HANDOFF_BRIEF.md`
3. Read `02_EMS_DESIGN_CONTEXT_BUNDLE.md`
4. Review screenshot references in `screenshots/SCREENSHOT_IMAGE_MANIFEST.md`
5. Review `sources/` folders only when deeper context is needed
